# 📰 News Sentiment Analyzer

A modern AI-powered web application that analyzes news content and classifies it into different categories such as Technology, Sports, Politics, Entertainment, Wellness, Travel, and Study.

## 🚀 Features

- 🔍 Analyze news text
- 🔗 Analyze article URLs
- 📸 Analyze images
- 📊 Category confidence scores
- 🎨 Modern responsive UI
- ⚡ Fast and lightweight

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Design
- NLP-based keyword analysis
